  • • 完全ローカル実行：センシティブなデータがクラウドに送信されないため、企業の機密情報も安全に処理
  • • オフライン対応：インターネット接続に依存せず、安定した動作環境を提供
  • • ネイティブパフォーマンス：ブラウザ制約を受けない高速な処理と豊富なシステム連携
  • • クロスプラットフォーム：Windows、macOS、Linuxで一貫したユーザー体験
  • • 独立した実行環境：ブラウザのセキュリティ制限やアップデートの影響を受けない安定動作

ユーザーマニュアル

AssIstant_Atelierの機能詳細と使用方法

🚀

アプリケーション概要

AssIstant_AtelierはElectron-Viteベースのデスクトップアプリケーションです。React、TypeScript、Chakra-UI、Tailwind CSSで構築され、Google AI (Gemini)のAPIを活用した特化型AIアシスタントプラットフォームです。

技術スタック

  • フレームワーク: Electron (electron-vite)
  • フロントエンド: React, TypeScript
  • UIライブラリ: Chakra-UI, Tailwind CSS
  • 国際化: react-i18next（日本語・英語対応）
💬

基本的な使い方

初回セットアップ

  1. アプリケーションを起動
  2. API設定からGoogle AI APIキーを設定
  3. アシスタントを作成して会話開始

チャット機能

  • マークダウン表示対応
  • メッセージのコピー・編集機能
  • ファイル添付（ドラッグ&ドロップ対応）
🤖

アシスタント管理

アシスタント作成

  • カスタムシステムプロンプト設定
  • 専門的なロール指示の設定
  • ナレッジベースの事前搭載
  • アシスタントの並び替え（ドラッグ&ドロップ）

管理機能

  • 複数アシスタントの作成・切り替え
  • アシスタントの編集・削除
  • アシスタント設定のエクスポート・インポート
⚡

オートアシスト機能

複雑なタスクを自動分解し、最適なアシスタントを選択して実行する高度な機能です。

主な特徴

  • タスクの自動分解と最適化
  • 複数アシスタントの連携処理
  • エージェントモード（自動実行）
  • Yes/No確認による安全な実行

使用方法

  1. オートアシストモードを有効化
  2. 複雑なタスクを入力
  3. システムが自動でタスクを分解
  4. 適切なアシスタントが順次実行
📁

ファイル処理機能

対応ファイル形式

  • PDF: 文書内容の読み取り・解析
  • CSV: データの読み込み・JSON変換
  • TXT: テキストファイルの処理
  • 画像: PNG、JPG、GIF等の解析

操作方法

  • ドラッグ&ドロップでファイル添付
  • AIによる内容理解と質問応答
💾

データ管理機能

エクスポート機能

  • 全データまたは選択したアシスタントのエクスポート
  • 設定情報の含める/含めないを選択可能

インポート機能

  • 既存アシスタントの置き換えまたは追加
  • バックアップからの復元

自動管理

  • 会話履歴の自動保存
  • 添付ファイルの自動管理
  • 不要ファイルの自動クリーンアップ
🌐

多言語対応

言語設定

  • 対応言語: 日本語、英語
  • 初回起動時の自動言語検出
  • 設定画面からの即座な言語切り替え
  • 設定の永続化（再起動後も維持）

完全翻訳

  • 全UI要素の翻訳対応
  • メニュー、ボタン、メッセージの多言語化
  • エラー表示の多言語対応
🎨

カスタマイズ機能

UI カスタマイズ

  • アプリ名の変更機能(アプリ名の左側のボタン)
  • ヘッダー画像の設定

レイアウト調整

  • リサイズ可能なサイドバー（280px-600px）
  • レスポンシブデザイン
  • ドラッグ&ドロップによるアシスタント移動
🔒

セキュリティ機能

データ保護

  • APIキーの暗号化保存
  • マシン固有の暗号化キーを使用
  • 機密情報の分離保存
  • プロキシ設定対応（企業環境）

ファイル検証

  • 添付ファイルの種類・内容検証
  • 安全なファイル処理
  • Electronセキュリティベストプラクティス準拠
